-------------------------------------------------------------
Flux distributions in central metabolism of human hepatocytes 
-------------------------------------------------------------

Model construction:
-------------------

1. The Hepatonet network structure and minimal flux distributions 
   for various target functions were obtained from an author of [1]

2. The flux distributions for aerobic regeneration of ATP and anaerobic regeneration of UTP
   were chosen as test cases.

3. Kinetic models were constructed, based on the hepatonet network structure, restricted to the
   flux distributions, by elasticity sampling (data in struct 'reference_state' in files .._reference_state).


Model files:
------------

1. Hepatonet_Anaerobic_UTP_Regeneration.xml                  SBML file
2. Hepatonet_Anaerobic_UTP_Regeneration.mat                  MATLAB .mat file (for MNT toolbox)
3. Hepatonet_Anaerobic_UTP_Regeneration_reference_state.mat  MATLAB .mat file (for MNT toolbox)
4. Hepatonet_Aerobic_ATP_Regeneration.xml                    SBML file
5. Hepatonet_Aerobic_ATP_Regeneration.mat                    MATLAB .mat file (for MNT toolbox)
6. Hepatonet_Aerobic_ATP_Regeneration_reference_state.mat    MATLAB .mat file (for MNT toolbox)


Reference:
----------

[1] C. Gille, C. Bölling, A. Hoppe, S. Bulik, S. Hoffmann, K. Hübner, A. Karlstädt, 
    R. Ganeshan, M. König, K. Rother, M. Weidlich, J. Behre, and H. Holzhütter (2010)
    HepatoNet1: a comprehensive metabolic reconstruction of the human hepatocyte 
    for the analysis of liver physiology. Molecular Systems Biology, 6:411.
